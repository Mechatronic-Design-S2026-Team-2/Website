Overview

Final architecture: leg-initiated locomotion from six independently phased Klann linkage legs, Jetson-hosted mapping/navigation, ESP32 motor/power management, and operator-supervised autonomy.

Locomotion: six Klann linkage leg modules, one RS485 servo + 50:1 gearbox per leg
Gait: alternating tripod phase relationships with startup contact-pose sequencing
Motor feedback: raw 17-bit single-turn motor encoder counts reconstructed on the Jetson through the gearbox ratio
Mapping: RealSense D415 + ORB-SLAM2 CUDA + ROS 2 UDP bridge + slam_toolbox occupancy grid
Navigation: Nav2/MPPI-oriented and teleop command paths using a common body-velocity interface
Safety: precharge/contactor sequencing, shared servo enable, stop-prioritized RS485 scheduling, and operator e-stop mode
