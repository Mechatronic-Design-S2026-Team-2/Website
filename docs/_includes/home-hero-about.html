About this project

Team 2 built a human-carrying Klann-linkage hexapod for AXIS Dance Company's Kinematic/Kinesthetic project. The class-final robot moved under its own legged locomotion, carried rider-scale loads, and integrated onboard mapping and ROS 2 control so that the robot could be operated as a stage mobility platform rather than just a mechanical prototype.

The final machine uses six independently commanded RS485 DC servo motors, each driving a 50:1 planetary gearbox and one Klann linkage. An ESP32 performs power sequencing, contactor/precharge control, servo enable, RS485 command writes, raw encoder sweeps, and micro-ROS telemetry. A Jetson Nano runs the ROS 2 Jazzy stack in Docker while ORB-SLAM2 CUDA and the RealSense D415 run natively on the Jetson host for GPU and camera-driver stability.

The project treats assistive mobility as performance media. The robot is designed to create visible, species-atypical motion: mechanically legged forward/backward travel, zero-radius turning, and operator-supervised navigation behavior that can be choreographed or controlled from off stage.

Payload>200 lb demonstrated without structural failure
Speed~0.7 m/s unloaded; ~0.44 m/s with rider
Accuracy target6 in stage-navigation tolerance
ComputeJetson Nano + ESP32 distributed stack
