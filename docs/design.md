    # System Design

    ## A) System requirements (specs + your derived constraints)
Create a small table or bullet list of:
- Physical constraints (size/weight/payload)
- Mobility requirements (directions, turning, speed range)
- Power/battery
- Safety requirements
- **Coolness factors** (your team-defined stretch features)

## B) Functional architecture
Add a block diagram showing:
- Major system functions
- Information flow (sensors → estimation → planning → control)
- Human/operator interface (if any)

## C) Cyberphysical architecture
List/diagram:
- Hardware components (compute, sensors, actuators, power)
- Software components (ROS nodes/modules, pipelines)
- Key interfaces (USB, I2C, UART, CAN, GPIO, etc.)

## D) System depictions & subsystem breakdown
- Full-system concept sketch / CAD / annotated photo
- Subsystem sections (mechanical, electrical, software/control)
- Brief alternatives considered and why you chose your approach
