The final class robot is a compact human-carrying hexapod built around a load-bearing chassis, a rider support surface, protected electronics, auxiliary support elements, and six Klann-linkage leg modules. The final report identifies the current design as successful for primary locomotion and load-bearing goals, while noting that auxiliary support elements still carry part of the practical stability burden.

Planned final CAD uploads should be placed at assets/designs/v5_final_assembly.glb for the full assembly and assets/designs/klann_final.glb for the final leg/linkage export. The gallery placeholders on the Media page will become interactive model viewers once those GLB files are committed.

Klann Linkage Assembly
{% include model-card.html wrap="false" file="/assets/designs/klann-leg.glb" caption="Earlier Klann-linkage GLB. Replace with /assets/designs/klann_final.glb when the final export is available." %}

The implemented robot uses one independently commanded RS485 servo motor per leg, not one mechanically coupled shaft per side. Each motor drives a TG-series 60 mm 50:1 planetary gearbox, which drives one Klann crank. This preserves the mechanically encoded stance/swing trajectory while giving software direct control over each leg phase.

The final software represents the leg as measured linkage geometry rather than a wheel radius approximation. For each output crank phase, the model returns foot position, foot velocity, and stance confidence. Runtime code can use precomputed lookup tables so that Jetson compute is spent on perception and navigation instead of repeated nonlinear linkage solves.
