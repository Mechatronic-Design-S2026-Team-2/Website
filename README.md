# Team 2 — Dance Robot Project Website (Template)

This repo is a **GitHub Pages** template site for the Mechatronic Design project website.

## Quick start (GitHub Pages)
1. Create a new **public** GitHub repo (or use an org repo), and upload everything from this folder.
2. In GitHub: **Settings → Pages**
3. Under **Build and deployment**:
   - **Source:** Deploy from a branch
   - **Branch:** `main`
   - **Folder:** `/docs`
4. Save. Your site will appear at a URL like:
   - `https://<username-or-org>.github.io/<repo>/`

## Editing
All pages live in `docs/`. Edit the Markdown (`.md`) files and commit.

## What’s included
- Required top-level pages: Problem, Design, Implementation, Performance, Management, Media, Team, Documents
- A Documents “workbook” area with folders for brainstorming, schematics, testing, software, presentations, ILRs
- Project management templates: schedule, parts list, issues log (CSV)
